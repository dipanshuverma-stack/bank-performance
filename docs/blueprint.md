# **App Name**: Dipanshu's Performance Elite

## Core Features:

- Intelligent Study Logic Tool: Generative AI acting as a tool to analyze reasoning and quants data, deciding which sub-topics from the Adda247 syllabus require focus based on error patterns.
- Adaptive To-Do Optimizer: Generative AI acting as a tool to automatically populate the daily to-do list with recommended chapters for Reasoning and Quants based on identified weak areas in performance logs.
- Unified Performance Dashboard: A modern centralized dashboard pulling test result data via integrations with Testbook or Oliveboard APIs.
- Precision Accuracy Timer: A stopwatch calibrated to record time-per-question and total chapter duration for performance mapping in reasoning and quants.
- Chapter-Wise Logger: Store and categorize accuracy percentages for Reasoning and Quants modules based on Adda247 curriculum.
- Visual Syllabus Tracker: Interactive checklist of the Adda247 bank exam syllabus for structured tracking of completion status.
- Daily Study Checklist: A dedicated daily task list for tracking preparation milestones and scheduled mock tests.
- Exam Countdown Tracker: A dynamic ticker displaying upcoming bank exam dates and the number of days remaining to maintain urgency.
- Daily Inspiration Hub: Generative AI acting as a tool to curate and display daily personalized motivational quotes specifically tailored for bank exam aspirants.

## Style Guidelines:

- Minimalist palette featuring 'Slate Grey' (#1E293B) and 'Pure White' (#FFFFFF) with 'Electric Indigo' (#6366F1) as the primary brand color.
- Accent colors: 'Emerald Green' (#10B981) for success metrics and 'Rose Red' (#EF4444) for error tracking.
- Modern sans-serif pairing: 'Satoshi' for headlines to provide a clean, premium feel and 'Plus Jakarta Sans' for readability in data-heavy sections.
- Clean, modular grid layout with significant whitespace and glassmorphism effects (frosted glass backgrounds) for cards.
- Custom-designed, minimalist stroke icons with a 2px weight to ensure a sharp, professional look.
- Fluid, spring-based transitions for data visualizations and smooth hover effects on interactive elements.
- Responsive bento-style cards that adapt seamlessly across devices while maintaining a high-end editorial feel.
